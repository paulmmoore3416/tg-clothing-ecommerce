## ğŸš€ Quick Start

### View Live Site
Simply visit: [https://paulmmoore3416.github.io/tg-clothing-ecommerce/](https://paulmmoore3416.github.io/tg-clothing-ecommerce/)

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/paulmmoore3416/tg-clothing-ecommerce.git
   cd tg-clothing-ecommerce
   ```

2. **Open in browser**
   ```bash
   # Using Python
   python3 -m http.server 8000

   # Or using Node.js
   npx http-server

   # Or just open index.html directly
   open index.html
   ```

3. **Access the site**
   - Open: http://localhost:8000
   - Browse products at: http://localhost:8000/products.html
